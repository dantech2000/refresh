# Release Notes - v0.1.7

## Features

### Dry Run Mode
- **`--dry-run` Flag**: Preview changes without executing them
- Shows what would happen during AMI updates without making actual changes

### Enhanced User Experience
- **Entertaining Spinner**: Added fun spinner with 10 humorous random messages during AWS API calls
- Messages cycle every 2 seconds to keep users entertained during data fetching

### Code Quality
- **Package Refactoring**: Broke up `main.go` into logical packages for better maintainability
- **Status Handling**: Switched to enums/struct hybrid for cleaner status management

## Security

- **oauth2 Module**: Security patch bump from v0.10.x to v0.27.0

## Usage Examples

```bash
# Preview AMI update without making changes
refresh update --cluster my-cluster --nodegroup my-nodegroup --dry-run

# List with entertaining spinner
refresh list --cluster my-cluster
```
