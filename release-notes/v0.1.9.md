# Release Notes - v0.1.9

## Features

### Man Page Installation
- **`refresh install-man`**: New command to install man pages locally
- Comprehensive CLI documentation accessible via `man refresh`

### Enhanced Documentation
- Improved help text and command descriptions
- Better inline documentation

### CI/CD Improvements
- **Claude Code Review**: Added AI-powered code review workflow
- **Claude PR Assistant**: Automated PR assistance workflow

## Bug Fixes

- Fixed linting errors in test files
- Addressed code review concerns with comprehensive tests

## Usage Examples

```bash
# Install man page
refresh install-man

# View man page
man refresh
```
