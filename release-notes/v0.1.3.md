# Release Notes - v0.1.3

## Features

### Initial Public Release
- **Tree-Style Output**: Switched from table format to tree-style output for better readability
- **Partial Name Matching**: Added fuzzy matching for cluster and nodegroup names
- **Version Command**: Added `refresh version` to display CLI version

### Developer Experience
- **Taskfile.yml**: Comprehensive task runner with development workflow commands
- **GoReleaser Integration**: Automated releases with Homebrew tap support
- **GitHub Actions**: Automated release workflow

### Code Quality
- **Google Go Best Practices**: Improved function naming conventions
- **Linter Fixes**: Resolved all linting issues

## Usage Examples

```bash
# Check version
refresh version

# List nodegroups with partial cluster name
refresh list --cluster dev
```
