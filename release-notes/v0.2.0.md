# Release Notes - v0.2.0

## Features

### Enhanced Cluster Operations
- **`describe cluster`**: Detailed cluster information display
- **`describe nodegroup`**: Comprehensive nodegroup details
- **`list addons`**: View installed EKS addons
- **`compare clusters`**: Compare configurations between clusters

- Service-oriented architecture for better maintainability
- Centralized AWS client management
- Enhanced error handling patterns
